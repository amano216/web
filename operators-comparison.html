言語別の特徴

JavaScript: == は型変換を行う等価演算子、=== は型変換を行わない厳密等価演算子

Python: is 演算子でオブジェクトの同一性を確認可能

Ruby: eql? と equal? メソッドで異なるレベルの等価性を確認

SQL: 標準では <> を使用、多くのDBMSで != もサポート

短絡評価（Short-circuit evaluation）

すべての言語で論理演算子は短絡評価を行います：

  • AND: 左辺が偽の場合、右辺は評価されない
  • OR: 左辺が真の場合、右辺は評価されない

重要な注意事項

  • JavaScript: == と === の違いに注意。通常は === を推奨
  • Python: インクリメント演算子 ++ は存在しない。i += 1 を使用
  • Ruby: and/or と &&/|| は優先順位が異なる
  • SQL: NULL の比較には IS NULL/IS NOT NULL を使用（= NULL は機能しない）
