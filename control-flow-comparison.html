条件分岐・制御構文比較表

JavaScript、Ruby、Pythonにおける条件分岐と制御構文の書き方を比較します。

基本的な if 文

構文要素 JavaScript Ruby Python
基本構文 if (条件) { 処理 } if 条件 then 処理 end if 条件: 処理
else if / elsif else if (条件) elsif 条件 elif 条件:
else else { 処理 } else 処理 end else: 処理
否定条件 if (!条件) unless 条件 if not 条件:

if文の例

JavaScript

let age = 20;

if (age >= 20) {
    console.log("成人です");
} else if (age >= 13) {
    console.log("ティーンエイジャーです");
} else {
    console.log("子供です");
}

Ruby

age = 20

if age >= 20
  puts "成人です"
elsif age >= 13
  puts "ティーンエイジャーです"
else
  puts "子供です"
end

Python

age = 20

if age >= 20:
    print("成人です")
elif age >= 13:
    print("ティーンエイジャーです")
else:
    print("子供です")

三項演算子・条件式

言語 構文 例
JavaScript 条件 ? 真の値 : 偽の値 age >= 20 ? "成人" : "未成年"
Ruby 条件 ? 真の値 : 偽の値 age >= 20 ? "成人" : "未成年"
Python 真の値 if 条件 else 偽の値 "成人" if age >= 20 else "未成年"

switch文 / case文

構文要素 JavaScript Ruby Python
キーワード switch / case case / when match / case (3.10+)
デフォルト default: else case _:
break必要性 必要 不要 不要

switch/case文の例

JavaScript

let day = "月曜日";

switch (day) {
    case "月曜日":
    case "火曜日":
    case "水曜日":
    case "木曜日":
    case "金曜日":
        console.log("平日です");
        break;
    case "土曜日":
    case "日曜日":
        console.log("週末です");
        break;
    default:
        console.log("不明な曜日");
}

Ruby

day = "月曜日"

case day
when "月曜日", "火曜日", "水曜日", "木曜日", "金曜日"
  puts "平日です"
when "土曜日", "日曜日"
  puts "週末です"
else
  puts "不明な曜日"
end

Python

# Python 3.10以降
day = "月曜日"

match day:
    case "月曜日" | "火曜日" | "水曜日" | "木曜日" | "金曜日":
        print("平日です")
    case "土曜日" | "日曜日":
        print("週末です")
    case _:
        print("不明な曜日")

# Python 3.9以前（辞書を使用）
weekday_type = {
    "月曜日": "平日", "火曜日": "平日",
    "水曜日": "平日", "木曜日": "平日",
    "金曜日": "平日", "土曜日": "週末",
    "日曜日": "週末"
}
print(weekday_type.get(day, "不明な曜日"))

ループ文（for文）

種類 JavaScript Ruby Python
基本for文 for (let i = 0; i < 10; i++) for i in 0..9 for i in range(10):
配列の反復 for (let item of array) array.each do |item| for item in array:
インデックス付き array.forEach((item, i) =>) array.each_with_index do |item, i| for i, item in enumerate(array):

for文の例

JavaScript

// 配列の反復
const fruits = ["りんご", "バナナ", "オレンジ"];

for (let fruit of fruits) {
    console.log(fruit);
}

// インデックス付き
fruits.forEach((fruit, index) => {
    console.log(`${index}: ${fruit}`);
});

Ruby

# 配列の反復
fruits = ["りんご", "バナナ", "オレンジ"]

fruits.each do |fruit|
  puts fruit
end

# インデックス付き
fruits.each_with_index do |fruit, index|
  puts "#{index}: #{fruit}"
end

Python

# 配列の反復
fruits = ["りんご", "バナナ", "オレンジ"]

for fruit in fruits:
    print(fruit)

# インデックス付き
for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")

while文

構文要素 JavaScript Ruby Python
基本構文 while (条件) { 処理 } while 条件 do 処理 end while 条件: 処理
do-while do { 処理 } while (条件) begin 処理 end while 条件 なし（whileで代替）
否定条件 while (!条件) until 条件 while not 条件:

while文の例

JavaScript

let count = 0;

while (count < 5) {
    console.log(`カウント: ${count}`);
    count++;
}

// do-while
let x = 0;
do {
    console.log(x);
    x++;
} while (x < 3);

Ruby

count = 0

while count < 5
  puts "カウント: #{count}"
  count += 1
end

# until（否定条件）
x = 0
until x >= 5
  puts x
  x += 1
end

Python

count = 0

while count < 5:
    print(f"カウント: {count}")
    count += 1

# do-whileの代替
x = 0
while True:
    print(x)
    x += 1
    if x >= 3:
        break

例外処理（try-catch）

構文要素 JavaScript Ruby Python
試行 try { } begin try:
エラー捕捉 catch (e) { } rescue => e except Exception as e:
最終処理 finally { } ensure finally:
エラー発生 throw new Error() raise raise Exception()

例外処理の例

JavaScript

try {
    let result = 10 / 0;
    if (!isFinite(result)) {
        throw new Error("ゼロ除算エラー");
    }
} catch (error) {
    console.error(`エラー: ${error.message}`);
} finally {
    console.log("処理完了");
}

Ruby

begin
  result = 10 / 0
rescue ZeroDivisionError => e
  puts "エラー: #{e.message}"
rescue => e
  puts "その他のエラー: #{e.message}"
ensure
  puts "処理完了"
end

Python

try:
    result = 10 / 0
except ZeroDivisionError as e:
    print(f"エラー: {e}")
except Exception as e:
    print(f"その他のエラー: {e}")
finally:
    print("処理完了")

演算 JavaScript Ruby Python

比較演算子

演算 JavaScript Ruby Python
等しい == / === == ==
等しくない != / !== != !=
より大きい > > >
より小さい < < <
以上 >= >= >=
以下 <= <= <=

重要な違い

  • JavaScript: ==は型変換を行い、===は厳密等価（型も値も同じ）
  • Ruby: elsifのスペル、unlessによる否定条件、case文でbreakが不要
  • Python: インデントが構文の一部、elifのスペル、match文は3.10以降

特殊な条件分岐パターン

ガード節（早期リターン）

JavaScript

function processUser(user) {
    if (!user) return null;
    if (!user.isActive) return null;
    if (user.age < 18) return null;
    
    // メイン処理
    return processValidUser(user);
}

Ruby

def process_user(user)
  return nil unless user
  return nil unless user.active?
  return nil if user.age < 18
  
  # メイン処理
  process_valid_user(user)
end

Python

def process_user(user):
    if not user:
        return None
    if not user.is_active:
        return None
    if user.age < 18:
        return None
    
    # メイン処理
    return process_valid_user(user)
