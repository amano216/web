CSS記法チートシート

CSSの記法を素早く参照できるクイックリファレンスです。

セレクタの完全ガイド

基本セレクタ

セレクタ 記法 説明・例
全称セレクタ * すべての要素を選択。
* { margin: 0; padding: 0; }
要素セレクタ element 指定したHTML要素を選択。
p { color: blue; }
h1 { font-size: 2rem; }
クラスセレクタ .class 指定したクラスを持つ要素を選択。
.container { width: 100%; }
.btn-primary { background: blue; }
IDセレクタ #id 指定したIDを持つ要素を選択。
#header { position: fixed; }
#main-nav { display: flex; }
属性セレクタ [attribute] 属性を持つ要素を選択。
[disabled] { opacity: 0.5; }
input[type="text"] { border: 1px solid #ccc; }
グループセレクタ A, B 複数のセレクタをカンマで区切る。
h1, h2, h3 { margin-top: 20px; }
.btn, .link { cursor: pointer; }

結合子（Combinators）

結合子 記法 説明・例
子孫セレクタ A B Aの中のすべてのBを選択（孫要素も含む）。
article p { line-height: 1.6; }
.container .item { padding: 10px; }
子セレクタ A > B Aの直接の子要素Bのみを選択。
ul > li { list-style: none; }
.nav > a { display: inline-block; }
隣接兄弟セレクタ A + B Aの直後にある兄弟要素Bを選択。
h1 + p { margin-top: 0; }
.active + li { font-weight: bold; }
一般兄弟セレクタ A ~ B Aの後にあるすべての兄弟要素Bを選択。
h2 ~ p { margin-left: 20px; }
.title ~ .content { color: gray; }

属性セレクタの詳細

属性セレクタ 記法 説明・例
存在 [attr] 属性が存在する要素。
[required] { border-color: red; }
[data-tooltip] { cursor: help; }
完全一致 [attr="value"] 属性値が完全に一致。
input[type="email"] { }
a[target="_blank"] { }
単語を含む [attr~="value"] 空白区切りの単語として含む。
[class~="active"] { }
[rel~="nofollow"] { }
前方一致 [attr^="value"] 属性値が指定値で始まる。
a[href^="https"] { }
[class^="col-"] { }
後方一致 [attr$="value"] 属性値が指定値で終わる。
a[href$=".pdf"] { }
img[src$=".jpg"] { }
部分一致 [attr*="value"] 属性値に指定値を含む。
a[href*="example"] { }
[class*="btn"] { }
言語コード [attr|="value"] 値またはハイフン付きで始まる。
[lang|="en"] { } /* en, en-US, en-GB等 */
[lang|="ja"] { }

疑似クラス（状態）

疑似クラス 記法 説明・例
ホバー :hover マウスが乗った時。
a:hover { text-decoration: underline; }
.btn:hover { opacity: 0.8; }
アクティブ :active クリック中。
button:active { transform: scale(0.95); }
フォーカス :focus フォーカスが当たった時。
input:focus { outline: 2px solid blue; }
訪問済み :visited 訪問済みのリンク。
a:visited { color: purple; }
チェック済み :checked チェックされた状態。
input:checked + label { font-weight: bold; }
無効 :disabled 無効化された要素。
button:disabled { opacity: 0.5; cursor: not-allowed; }
有効 :enabled 有効な要素。
input:enabled { background: white; }
空 :empty 子要素やテキストを持たない。
p:empty { display: none; }
ターゲット :target URLのハッシュと一致するID。
#section1:target { background: yellow; }

疑似クラス（構造）

疑似クラス 記法 説明・例
最初の子 :first-child 親要素の最初の子要素。
li:first-child { font-weight: bold; }
最後の子 :last-child 親要素の最後の子要素。
p:last-child { margin-bottom: 0; }
n番目の子 :nth-child(n) n番目の子要素。
tr:nth-child(odd) { background: #f0f0f0; }
li:nth-child(3n) { color: red; }
後ろからn番目 :nth-last-child(n) 後ろからn番目の子要素。
li:nth-last-child(2) { font-style: italic; }
特定タイプの最初 :first-of-type 同じタイプの最初の要素。
p:first-of-type { text-indent: 2em; }
特定タイプの最後 :last-of-type 同じタイプの最後の要素。
h2:last-of-type { margin-bottom: 30px; }
唯一の子 :only-child 親要素の唯一の子要素。
p:only-child { text-align: center; }
否定 :not(selector) 指定セレクタに一致しない要素。
input:not([type="submit"]) { }
li:not(:last-child) { }

疑似要素

疑似要素 記法 説明・例
前に挿入 ::before 要素の前に内容を挿入。
.required::before {
content: "*";
color: red;
}
後に挿入 ::after 要素の後に内容を挿入。
.external-link::after {
content: " ↗";
}
最初の行 ::first-line 最初の行のスタイル。
p::first-line {
font-weight: bold;
}
最初の文字 ::first-letter 最初の文字のスタイル。
p::first-letter {
font-size: 2em;
float: left;
}
選択部分 ::selection 選択されたテキスト。
::selection {
background: yellow;
color: black;
}
プレースホルダー ::placeholder 入力欄のプレースホルダー。
input::placeholder {
color: #999;
font-style: italic;
}

値と単位

長さの単位

単位 名前 説明・例
px Pixels ピクセル（絶対単位）。
width: 200px;
font-size: 16px;
em Em 親要素のフォントサイズに対する相対値。
padding: 1.5em; /* 親が16pxなら24px */
margin: 2em;
rem Root Em ルート要素（html）のフォントサイズに対する相対値。
font-size: 1.5rem; /* htmlが16pxなら24px */
width: 20rem;
% Percent 親要素に対する割合。
width: 50%;
font-size: 120%;
vw Viewport Width ビューポート幅の1%。
width: 100vw; /* 画面幅いっぱい */
font-size: 5vw;
vh Viewport Height ビューポート高さの1%。
height: 100vh; /* 画面高さいっぱい */
min-height: 50vh;
vmin Viewport Minimum vwとvhの小さい方。
width: 50vmin;
height: 50vmin; /* 正方形になる */
vmax Viewport Maximum vwとvhの大きい方。
font-size: 4vmax;
ch Character 文字「0」の幅。
width: 60ch; /* 約60文字分の幅 */
ex Ex Height 小文字「x」の高さ。
line-height: 3ex;

色の指定方法

形式 記法 説明・例
名前付き色 colorname 定義済みの色名。
color: red;
background: skyblue;
border-color: transparent;
16進数（6桁） #RRGGBB 赤緑青を16進数で。
color: #ff0000; /* 赤 */
background: #336699;
16進数（3桁） #RGB 短縮形（各桁が2回繰り返される）。
color: #f00; /* #ff0000と同じ */
background: #369; /* #336699と同じ */
16進数（8桁） #RRGGBBAA アルファチャンネル付き。
color: #ff000080; /* 半透明の赤 */
RGB rgb(R, G, B) 赤緑青を0-255で指定。
color: rgb(255, 0, 0);
background: rgb(51, 102, 153);
RGBA rgba(R, G, B, A) アルファチャンネル付き（0-1）。
color: rgba(255, 0, 0, 0.5);
background: rgba(0, 0, 0, 0.8);
HSL hsl(H, S%, L%) 色相・彩度・明度。
color: hsl(0, 100%, 50%); /* 赤 */
background: hsl(210, 50%, 40%);
HSLA hsla(H, S%, L%, A) HSLにアルファチャンネル追加。
color: hsla(0, 100%, 50%, 0.5);
currentColor currentColor 現在の文字色を継承。
border: 1px solid currentColor;

CSS関数

関数 記法 説明・例
calc() calc(expression) 計算式を使用。
width: calc(100% - 20px);
height: calc(100vh - 80px);
font-size: calc(1rem + 0.5vw);
var() var(--name, fallback) CSS変数を使用。
color: var(--primary-color);
padding: var(--spacing, 10px);
min() min(value1, value2, ...) 最小値を選択。
width: min(100%, 800px);
font-size: min(2rem, 5vw);
max() max(value1, value2, ...) 最大値を選択。
width: max(300px, 50%);
height: max(100px, 10vh);
clamp() clamp(min, preferred, max) 範囲内に収める。
font-size: clamp(16px, 4vw, 24px);
width: clamp(200px, 50%, 800px);
url() url(path) リソースのURL。
background-image: url('image.jpg');
content: url('icon.svg');
linear-gradient() linear-gradient(direction, color-stops) 線形グラデーション。
background: linear-gradient(to right, red, blue);
background: linear-gradient(45deg, #000 0%, #fff 100%);
radial-gradient() radial-gradient(shape size at position, color-stops) 円形グラデーション。
background: radial-gradient(circle, red, yellow, green);
background: radial-gradient(ellipse at center, #000, #fff);
conic-gradient() conic-gradient(from angle at position, color-stops) 円錐グラデーション。
background: conic-gradient(red, yellow, green);
background: conic-gradient(from 90deg, red, blue);

メディアクエリ

メディア特性 記法 説明・例
幅 width, min-width, max-width
@media (max-width: 768px) { /* タブレット以下 */ }
@media (min-width: 1024px) { /* デスクトップ以上 */ }
@media (width: 800px) { /* ちょうど800px */ }
高さ height, min-height, max-height
@media (min-height: 600px) { }
@media (max-height: 480px) { }
画面の向き orientation
@media (orientation: portrait) { /* 縦向き */ }
@media (orientation: landscape) { /* 横向き */ }
解像度 resolution
@media (min-resolution: 2dppx) { /* Retina */ }
@media (min-resolution: 300dpi) { }
アスペクト比 aspect-ratio
@media (aspect-ratio: 16/9) { }
@media (min-aspect-ratio: 4/3) { }
