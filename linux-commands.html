Linuxコマンド一覧

Linuxシステムで使用される主要なコマンドを、カテゴリ別にまとめました。

ファイル・ディレクトリ操作

基本的なファイル操作

コマンド 英語の意味 (Meaning) 日本語の意味・用途
ls List ディレクトリの内容を一覧表示する。-laで詳細表示
cd Change Directory ディレクトリを移動する。cd ~でホームへ
pwd Print Working Directory 現在のディレクトリパスを表示する
mkdir Make Directory 新しいディレクトリを作成する。-pで親も作成
rmdir Remove Directory 空のディレクトリを削除する
touch Touch 空ファイルの作成またはタイムスタンプ更新
cp Copy ファイルやディレクトリをコピーする。-rで再帰的
mv Move ファイルの移動・名前変更を行う
rm Remove ファイルを削除する。-rfで強制的に再帰削除
ln Link リンクを作成する。-sでシンボリックリンク

ファイル内容の表示・編集

コマンド 英語の意味 (Meaning) 日本語の意味・用途
cat Concatenate ファイルの内容を表示・連結する
less Less ファイルをページング表示する。qで終了
more More ファイルをページング表示する（基本的な機能）
head Head ファイルの先頭部分を表示。-n 20で20行
tail Tail ファイルの末尾を表示。-fでリアルタイム追跡
nano Nano シンプルなテキストエディタ
vim Vi Improved 高機能テキストエディタ。iで挿入、:wqで保存終了
echo Echo テキストを表示する。>でファイルに出力

ファイル検索・情報取得

検索コマンド

コマンド 英語の意味 (Meaning) 日本語の意味・用途
find Find ファイルを検索する。find . -name "*.txt"
grep Global Regular Expression Print パターンマッチングで検索。-rで再帰、-iで大小文字無視
which Which コマンドの実行ファイルパスを表示
whereis Where Is バイナリ、ソース、マニュアルの場所を検索
locate Locate データベースからファイルを高速検索
updatedb Update Database locateコマンド用のデータベースを更新

ファイル情報・統計

コマンド 英語の意味 (Meaning) 日本語の意味・用途
file File ファイルの種類を判定する
stat Statistics ファイルの詳細情報を表示
wc Word Count 行数・単語数・バイト数を数える。-lで行数のみ
du Disk Usage ディスク使用量を表示。-hで人間が読みやすい形式
df Disk Free ディスクの空き容量を表示。-hで読みやすく

プロセス管理

プロセスの確認・制御

コマンド 英語の意味 (Meaning) 日本語の意味・用途
ps Process Status 実行中のプロセスを表示。ps auxで全プロセス
top Top プロセスをリアルタイムで監視。qで終了
htop H Top topの改良版。より見やすいプロセス監視
kill Kill プロセスを終了する。kill -9 PIDで強制終了
killall Kill All プロセス名で全て終了する
jobs Jobs バックグラウンドジョブを表示
bg Background ジョブをバックグラウンドで実行
fg Foreground ジョブをフォアグラウンドに移動
nohup No Hang Up ログアウト後もコマンドを継続実行

ユーザー・権限管理

ユーザー操作

コマンド 英語の意味 (Meaning) 日本語の意味・用途
whoami Who Am I 現在のユーザー名を表示
id Identity ユーザーIDとグループIDを表示
su Switch User 別のユーザーに切り替える。su -で環境も切替
sudo Super User Do 管理者権限でコマンドを実行
useradd User Add 新しいユーザーを追加する
userdel User Delete ユーザーを削除する
passwd Password パスワードを変更する
groups Groups ユーザーが所属するグループを表示

権限管理

コマンド 英語の意味 (Meaning) 日本語の意味・用途
chmod Change Mode ファイルの権限を変更。chmod 755 file
chown Change Owner ファイルの所有者を変更。chown user:group file
chgrp Change Group ファイルのグループを変更
umask User Mask デフォルトのファイル権限マスクを設定

ネットワーク関連

ネットワーク確認・設定

コマンド 英語の意味 (Meaning) 日本語の意味・用途
ping Ping ネットワーク接続を確認。ping google.com
ifconfig Interface Configuration ネットワークインターフェースの設定・表示
ip Internet Protocol IPアドレスやルーティングを管理。ip addr show
netstat Network Statistics ネットワーク接続状況を表示。-tulnでポート確認
ss Socket Statistics ソケット統計情報を表示（netstatの代替）
curl Client URL URLからデータを取得。-Oでダウンロード
wget Web Get ファイルをダウンロード。-cで再開可能
ssh Secure Shell リモートサーバーに安全に接続
scp Secure Copy SSH経由でファイルをコピー

システム情報・管理

システム情報

コマンド 英語の意味 (Meaning) 日本語の意味・用途
uname Unix Name システム情報を表示。-aで全情報
hostname Host Name ホスト名を表示・設定する
date Date 現在の日時を表示・設定
uptime Up Time システムの稼働時間を表示
free Free メモリ使用状況を表示。-hで読みやすく
lscpu List CPU CPU情報を表示
lsblk List Block ブロックデバイス（ディスク）の一覧
mount Mount ファイルシステムをマウント・表示
umount Unmount ファイルシステムをアンマウント

パッケージ管理（Debian/Ubuntu系）

コマンド 英語の意味 (Meaning) 日本語の意味・用途
apt Advanced Package Tool パッケージ管理ツール。apt updateで更新
apt-get APT Get 従来のパッケージ管理コマンド
dpkg Debian Package 低レベルパッケージ管理。-iでインストール
snap Snap Snapパッケージの管理

パッケージ管理（RedHat/CentOS系）

コマンド 英語の意味 (Meaning) 日本語の意味・用途
yum Yellowdog Updater Modified パッケージ管理ツール。yum install
dnf Dandified YUM yumの後継パッケージマネージャー
rpm RPM Package Manager RPMパッケージの管理。-ivhでインストール

アーカイブ・圧縮

圧縮・展開コマンド

コマンド 英語の意味 (Meaning) 日本語の意味・用途
tar Tape Archive アーカイブの作成・展開。-czfで圧縮作成、-xzfで展開
gzip GNU Zip ファイルを圧縮。-dで展開
gunzip GNU Unzip gzipファイルを展開
zip Zip ZIPアーカイブを作成。-rで再帰的
unzip Unzip ZIPアーカイブを展開
bzip2 Bzip2 高圧縮率で圧縮。-dで展開
xz XZ 最高圧縮率で圧縮。-dで展開

テキスト処理

テキスト操作・フィルタ

コマンド 英語の意味 (Meaning) 日本語の意味・用途
sed Stream Editor ストリームエディタ。s/old/new/gで置換
awk AWK パターン処理言語。テキスト処理に便利
cut Cut 行から特定の部分を切り出す。-dで区切り文字
sort Sort 行をソートする。-nで数値順、-rで逆順
uniq Unique 重複行を削除。-cでカウント表示
tr Translate 文字を変換・削除。tr 'a-z' 'A-Z'で大文字化
tee Tee 標準出力とファイルに同時出力
xargs Extended Arguments 標準入力を引数に変換してコマンド実行

シェル・環境変数

シェル操作

コマンド 英語の意味 (Meaning) 日本語の意味・用途
bash Bourne Again Shell Bashシェルを起動
sh Shell 標準シェルを起動
source Source スクリプトを現在のシェルで実行。.も同じ
export Export 環境変数を設定・エクスポート
env Environment 環境変数を表示
alias Alias コマンドのエイリアスを作成
history History コマンド履歴を表示。!番号で再実行
which Which コマンドのパスを表示
type Type コマンドの種類を表示

ユーティリティ
