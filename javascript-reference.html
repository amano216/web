JavaScriptメソッド・プロパティ一覧

JavaScriptでよく使用されるメソッドとプロパティを、カテゴリ別にまとめました。

文字列（String）メソッド

基本的な文字列操作

メソッド・プロパティ 英語の意味 (Meaning) 日本語の意味・用途
length Length 文字列の長さを取得する。文字数をカウント
charAt(index) Character At 指定位置の文字を取得する。インデックスは0から開始
charCodeAt(index) Character Code At 指定位置の文字のUnicodeコードを取得する
concat(str) Concatenate 文字列を連結する。複数の文字列を結合
indexOf(searchValue) Index Of 文字列を検索して最初の位置を返す。見つからない場合は-1
lastIndexOf(searchValue) Last Index Of 文字列を後ろから検索して位置を返す
includes(searchString) Includes 文字列が含まれているかを判定する。true/falseを返す
startsWith(searchString) Starts With 文字列が指定文字で始まるかを判定する
endsWith(searchString) Ends With 文字列が指定文字で終わるかを判定する

文字列の変換・整形

メソッド 英語の意味 (Meaning) 日本語の意味・用途
toLowerCase() To Lower Case すべて小文字に変換する
toUpperCase() To Upper Case すべて大文字に変換する
trim() Trim 前後の空白を削除する
trimStart() Trim Start 先頭の空白を削除する
trimEnd() Trim End 末尾の空白を削除する
padStart(length, padString) Pad Start 文字列の先頭を指定文字で埋める
padEnd(length, padString) Pad End 文字列の末尾を指定文字で埋める
repeat(count) Repeat 文字列を指定回数繰り返す

文字列の分割・抽出

メソッド 英語の意味 (Meaning) 日本語の意味・用途
slice(start, end) Slice 文字列の一部を抽出する。開始位置から終了位置まで
substring(start, end) Substring 文字列の一部を抽出する。負の値は0として扱う
substr(start, length) Substr (deprecated) 指定位置から指定長さの文字列を抽出（非推奨）
split(separator) Split 文字列を分割して配列にする。区切り文字で分割
replace(pattern, replacement) Replace 文字列を置換する。最初の一致のみ
replaceAll(pattern, replacement) Replace All すべての一致を置換する
match(regexp) Match 正規表現にマッチする部分を取得する
search(regexp) Search 正規表現で検索して位置を返す

配列（Array）メソッド

配列の基本操作

メソッド・プロパティ 英語の意味 (Meaning) 日本語の意味・用途
length Length 配列の要素数を取得・設定する
push(element) Push 配列の末尾に要素を追加する
pop() Pop 配列の末尾から要素を削除して返す
unshift(element) Unshift 配列の先頭に要素を追加する
shift() Shift 配列の先頭から要素を削除して返す
concat(array) Concatenate 配列を結合する。元の配列は変更しない
join(separator) Join 配列を文字列に変換する。区切り文字を指定可能
reverse() Reverse 配列の順序を反転する。元の配列を変更
sort(compareFunction) Sort 配列をソートする。比較関数を指定可能

配列の検索・判定

メソッド 英語の意味 (Meaning) 日本語の意味・用途
indexOf(searchElement) Index Of 要素を検索して最初の位置を返す
lastIndexOf(searchElement) Last Index Of 要素を後ろから検索して位置を返す
includes(searchElement) Includes 要素が含まれているかを判定する
find(callback) Find 条件に合う最初の要素を返す
findIndex(callback) Find Index 条件に合う最初の要素の位置を返す
every(callback) Every すべての要素が条件を満たすか判定する
some(callback) Some いずれかの要素が条件を満たすか判定する

配列の変換・操作

メソッド 英語の意味 (Meaning) 日本語の意味・用途
slice(start, end) Slice 配列の一部を抽出する。元の配列は変更しない
splice(start, deleteCount, items) Splice 配列の要素を削除・追加する。元の配列を変更
map(callback) Map 各要素を変換して新しい配列を作成する
filter(callback) Filter 条件に合う要素だけの新しい配列を作成する
reduce(callback, initialValue) Reduce 配列を単一の値に集約する。累積処理
forEach(callback) For Each 各要素に対して処理を実行する。返り値なし
flat(depth) Flat ネストされた配列を平坦化する
flatMap(callback) Flat Map mapとflatを同時に実行する

数値（Number）メソッド

数値の変換・判定

メソッド・プロパティ 英語の意味 (Meaning) 日本語の意味・用途
Number.parseInt(string) Parse Integer 文字列を整数に変換する
Number.parseFloat(string) Parse Float 文字列を浮動小数点数に変換する
Number.isInteger(value) Is Integer 整数かどうかを判定する
Number.isNaN(value) Is Not a Number NaN（非数）かどうかを判定する
Number.isFinite(value) Is Finite 有限数かどうかを判定する
toFixed(digits) To Fixed 小数点以下の桁数を指定して文字列に変換
toPrecision(precision) To Precision 有効桁数を指定して文字列に変換
toString(radix) To String 数値を文字列に変換。基数を指定可能

Math オブジェクト

数学的計算

メソッド・プロパティ 英語の意味 (Meaning) 日本語の意味・用途
Math.PI Pi 円周率（π）の値。約3.14159
Math.abs(x) Absolute 絶対値を返す。負の数を正にする
Math.ceil(x) Ceiling 切り上げる。小数点以下を切り上げて整数に
Math.floor(x) Floor 切り捨てる。小数点以下を切り捨てて整数に
Math.round(x) Round 四捨五入する。最も近い整数に丸める
Math.max(...values) Maximum 最大値を返す。複数の値から最大を選択
Math.min(...values) Minimum 最小値を返す。複数の値から最小を選択
Math.pow(base, exponent) Power 累乗を計算する。baseのexponent乗
Math.sqrt(x) Square Root 平方根を計算する
Math.random() Random 0以上1未満の乱数を生成する

日付（Date）メソッド

日付の取得

メソッド 英語の意味 (Meaning) 日本語の意味・用途
getFullYear() Get Full Year 4桁の年を取得する（例：2024）
getMonth() Get Month 月を取得する（0-11、0が1月）
getDate() Get Date 日を取得する（1-31）
getDay() Get Day 曜日を取得する（0-6、0が日曜）
getHours() Get Hours 時を取得する（0-23）
getMinutes() Get Minutes 分を取得する（0-59）
getSeconds() Get Seconds 秒を取得する（0-59）
getMilliseconds() Get Milliseconds ミリ秒を取得する（0-999）
getTime() Get Time 1970年1月1日からのミリ秒を取得する

日付の設定

メソッド 英語の意味 (Meaning) 日本語の意味・用途
setFullYear(year) Set Full Year 年を設定する
setMonth(month) Set Month 月を設定する（0-11）
setDate(date) Set Date 日を設定する（1-31）
setHours(hours) Set Hours 時を設定する（0-23）
setMinutes(minutes) Set Minutes 分を設定する（0-59）
setSeconds(seconds) Set Seconds 秒を設定する（0-59）
setTime(milliseconds) Set Time 1970年1月1日からのミリ秒で日時を設定

非同期処理

Promise・async/await

メソッド・キーワード 英語の意味 (Meaning) 日本語の意味・用途
then(onFulfilled) Then Promiseが成功した時の処理を登録する
catch(onRejected) Catch Promiseが失敗した時の処理を登録する
finally(onFinally) Finally 成功・失敗に関わらず実行する処理を登録
Promise.all(promises) Promise All すべてのPromiseが完了するのを待つ
Promise.race(promises) Promise Race 最初に完了したPromiseの結果を返す
async Asynchronous 非同期関数を定義する。Promiseを返す関数
await Await Promiseの完了を待つ。async関数内でのみ使用
