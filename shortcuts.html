プログラミング記法 & ショートカットキー チートシート

JavaScript基本記法と作業効率を劇的に向上させる必須ショートカット集

JavaScript 基本記法

記法 英語の読み方 日本語の読み方と使い方
console.log() console log (コンソール ログ) コンソールログ。デバッグ用の値出力。開発者ツールのコンソールに表示
let let (レット) レット。再代入可能な変数を宣言。ブロックスコープ
const constant (コンスタント) コンスト。再代入不可能な定数を宣言。ブロックスコープ
var variable (ヴァリアブル) ヴァー。古い変数宣言。関数スコープ（非推奨）
function function (ファンクション) ファンクション。関数を定義。処理をまとめて再利用可能にする
=> arrow function (アロー ファンクション) アロー関数。簡潔な関数記法。thisの挙動が通常の関数と異なる
if if (イフ) イフ。条件分岐。条件が真の場合に処理を実行
else else (エルス) エルス。ifの条件が偽の場合に実行する処理
for for (フォー) フォー。繰り返し処理。指定回数または条件でループ
while while (ホワイル) ホワイル。条件が真の間、繰り返し処理
return return (リターン) リターン。関数から値を返す。関数の実行を終了
import import (インポート) インポート。他のモジュールから機能を読み込む
export export (エクスポート) エクスポート。モジュールから機能を外部に公開
class class (クラス) クラス。オブジェクト指向のクラスを定義
new new (ニュー) ニュー。クラスからインスタンスを生成
this this (ディス) ディス。現在のコンテキストのオブジェクトを参照
try...catch try catch (トライ キャッチ) トライキャッチ。エラー処理。例外をキャッチして処理
async asynchronous (エイシンク) エイシンク。非同期関数を定義。Promiseを返す
await await (アウェイト) アウェイト。非同期処理の完了を待つ。async関数内でのみ使用
null null (ヌル) ヌル。値が存在しないことを明示的に示す
undefined undefined (アンディファインド) アンディファインド。値が未定義の状態
true / false true false (トゥルー フォルス) トゥルー/フォルス。真偽値。条件判定に使用
=== strict equality (ストリクト イコーリティ) 厳密等価演算子。型と値が両方一致する場合にtrue
!== strict inequality (ストリクト インイコーリティ) 厳密不等価演算子。型または値が異なる場合にtrue
&& logical AND (ロジカル アンド) 論理積。両方の条件が真の場合にtrue
|| logical OR (ロジカル オア) 論理和。どちらかの条件が真の場合にtrue
! logical NOT (ロジカル ノット) 論理否定。真偽値を反転
... spread operator (スプレッド オペレーター) スプレッド演算子。配列やオブジェクトを展開
?: ternary operator (ターナリー オペレーター) 三項演算子。条件式 ? 真の場合 : 偽の場合

💡 効率化のヒント
  • 毎日1つずつ新しいショートカットを覚えて習慣化しましょう
  • よく使う操作から優先的に覚えると効果的です
  • マウスを使いたくなったら、そのショートカットを調べる習慣をつけましょう
  • エディタやツールの設定でカスタムショートカットを作成することも可能です
