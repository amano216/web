Rubyメソッド一覧

Rubyの主要なクラスとメソッドを、カテゴリ別にまとめました。

文字列（String）メソッド

文字列の操作と変換

メソッド 英語の意味 (Meaning) 日本語の意味・用途
length / size Length / Size 文字列の長さを返す
upcase Uppercase すべて大文字に変換。"hello".upcase #=> "HELLO"
downcase Downcase すべて小文字に変換。"HELLO".downcase #=> "hello"
capitalize Capitalize 最初の文字を大文字に。"hello".capitalize #=> "Hello"
reverse Reverse 文字列を逆順にする。"hello".reverse #=> "olleh"
strip Strip 前後の空白を削除。" hello ".strip #=> "hello"
chomp Chomp 末尾の改行を削除。"hello\n".chomp #=> "hello"
chop Chop 末尾の1文字を削除。"hello".chop #=> "hell"
gsub(pattern, replacement) Global Substitute パターンにマッチする部分をすべて置換
sub(pattern, replacement) Substitute 最初にマッチした部分を置換

文字列の検索と判定

メソッド 英語の意味 (Meaning) 日本語の意味・用途
include?(substring) Include 指定の文字列を含むか。"hello".include?("ell") #=> true
start_with?(prefix) Start With 指定の文字列で始まるか。"hello".start_with?("he") #=> true
end_with?(suffix) End With 指定の文字列で終わるか。"hello".end_with?("lo") #=> true
empty? Empty 空文字列か。"".empty? #=> true
match(pattern) Match 正規表現にマッチするか確認
index(substring) Index 部分文字列の位置を返す。見つからなければnil
scan(pattern) Scan パターンにマッチする部分をすべて配列で返す

文字列の分割と結合

メソッド 英語の意味 (Meaning) 日本語の意味・用途
split(delimiter) Split 区切り文字で分割。"a,b,c".split(",") #=> ["a", "b", "c"]
join(separator) Join 配列を文字列に結合（配列のメソッド）
concat(string) Concatenate 文字列を連結（破壊的）
+ Plus 文字列を連結（非破壊的）。"hello" + " world"
<< Append 文字列を追加（破壊的）。str << " world"
* Multiply 文字列を繰り返す。"ha" * 3 #=> "hahaha"

配列（Array）メソッド

要素の追加と削除

メソッド 英語の意味 (Meaning) 日本語の意味・用途
push(item) / << Push 末尾に要素を追加。arr << 5
pop Pop 末尾の要素を削除して返す
unshift(item) Unshift 先頭に要素を追加
shift Shift 先頭の要素を削除して返す
insert(index, item) Insert 指定位置に要素を挿入
delete(item) Delete 指定の値と一致する要素をすべて削除
delete_at(index) Delete At 指定位置の要素を削除
clear Clear すべての要素を削除

配列の検索と選択

メソッド 英語の意味 (Meaning) 日本語の意味・用途
first / first(n) First 最初の要素（またはn個）を返す
last / last(n) Last 最後の要素（またはn個）を返す
include?(item) Include 指定の要素を含むか。[1,2,3].include?(2) #=> true
find / detect Find / Detect 条件に一致する最初の要素を返す
select / filter Select / Filter 条件に一致する要素をすべて返す
reject Reject 条件に一致しない要素を返す
grep(pattern) Grep パターンにマッチする要素を返す

配列の変換と操作

メソッド 英語の意味 (Meaning) 日本語の意味・用途
map / collect Map / Collect 各要素を変換した新しい配列を作成
each Each 各要素に対して処理を実行
each_with_index Each With Index インデックス付きで各要素を処理
sort Sort 要素をソート。[3,1,2].sort #=> [1,2,3]
reverse Reverse 要素を逆順にする。[1,2,3].reverse #=> [3,2,1]
shuffle Shuffle 要素をランダムに並び替える
uniq Unique 重複を削除。[1,2,2,3].uniq #=> [1,2,3]
flatten Flatten 多次元配列を1次元に。[[1,2],[3,4]].flatten #=> [1,2,3,4]
compact Compact nilを削除。[1,nil,2,nil,3].compact #=> [1,2,3]

ハッシュ（Hash）メソッド

要素の取得と設定

メソッド 英語の意味 (Meaning) 日本語の意味・用途
[key] Get キーに対応する値を取得
[key] = value Set キーに値を設定
fetch(key) Fetch キーの値を取得（存在しなければエラー）
keys Keys すべてのキーを配列で返す
values Values すべての値を配列で返す
has_key? / key? Has Key 指定のキーが存在するか
has_value? / value? Has Value 指定の値が存在するか

ハッシュの操作

メソッド 英語の意味 (Meaning) 日本語の意味・用途
merge(other_hash) Merge 別のハッシュと結合（非破壊的）
merge!(other_hash) Merge! 別のハッシュと結合（破壊的）
delete(key) Delete 指定のキーと値を削除
clear Clear すべての要素を削除
each Each 各キーと値のペアに対して処理
select Select 条件に一致するペアを選択
reject Reject 条件に一致しないペアを選択
invert Invert キーと値を入れ替える

数値（Numeric）メソッド

数値の操作と判定

メソッド 英語の意味 (Meaning) 日本語の意味・用途
abs Absolute 絶対値を返す。-5.abs #=> 5
round Round 四捨五入。3.14.round #=> 3
ceil Ceiling 切り上げ。3.14.ceil #=> 4
floor Floor 切り捨て。3.14.floor #=> 3
even? Even 偶数か。4.even? #=> true
odd? Odd 奇数か。3.odd? #=> true
zero? Zero ゼロか。0.zero? #=> true
positive? Positive 正の数か。5.positive? #=> true
negative? Negative 負の数か。-5.negative? #=> true
times Times 指定回数繰り返す。3.times { puts "hello" }

範囲（Range）メソッド

範囲の操作

メソッド 英語の意味 (Meaning) 日本語の意味・用途
to_a To Array 配列に変換。(1..5).to_a #=> [1,2,3,4,5]
include? / cover? Include / Cover 値が範囲内か。(1..10).include?(5) #=> true
each Each 範囲内の各値に対して処理
step(n) Step n個おきに処理。(0..10).step(2) { |i| puts i }
first / begin First / Begin 範囲の最初の値
last / end Last / End 範囲の最後の値

ファイル（File）クラス

ファイル操作

メソッド 英語の意味 (Meaning) 日本語の意味・用途
File.open(path, mode) Open ファイルを開く。modeは"r", "w", "a"など
File.read(path) Read ファイル全体を読み込む
File.write(path, content) Write ファイルに書き込む
File.exist?(path) Exist ファイルが存在するか
File.size(path) Size ファイルサイズを取得
File.delete(path) Delete ファイルを削除
File.rename(old, new) Rename ファイル名を変更
File.directory?(path) Directory ディレクトリか判定

時刻（Time）クラス

時刻の操作

メソッド 英語の意味 (Meaning) 日本語の意味・用途
Time.now Now 現在時刻を取得
Time.new(year, month, day) New 指定日時のTimeオブジェクトを作成
strftime(format) String Format Time 指定形式で文字列化。time.strftime("%Y-%m-%d")
year / month / day Year / Month / Day 年/月/日を取得
hour / min / sec Hour / Minute / Second 時/分/秒を取得
wday Weekday 曜日を数値で取得（0:日曜〜6:土曜）
+ / - Plus / Minus 秒数を加算/減算。time + 3600（1時間後）

Enumerable（列挙可能）モジュール

集約と変換

メソッド 英語の意味 (Meaning) 日本語の意味・用途
reduce / inject Reduce / Inject 要素を集約。[1,2,3].reduce(:+) #=> 6
all? All すべて条件を満たすか。[2,4,6].all?(&:even?) #=> true
any? Any いずれか条件を満たすか。[1,2,3].any?(&:even?) #=> true
none? None どれも条件を満たさないか
count Count 要素数または条件に一致する数を返す
max / min Maximum / Minimum 最大値/最小値を返す
max_by / min_by Max By / Min By ブロックの結果が最大/最小の要素を返す
group_by Group By ブロックの結果でグループ化
partition Partition 条件で真と偽の2つの配列に分割
