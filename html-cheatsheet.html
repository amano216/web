HTML記法チートシート

HTMLの基本的な記法と構文ルールのクイックリファレンスです。

HTML文書の基本構造

最小限のHTML5テンプレート

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ページタイトル</title>
</head>
<body>
    <!-- コンテンツをここに記述 -->
</body>
</html>

完全なHTML5テンプレート

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="ページの説明">
    <meta name="keywords" content="キーワード1, キーワード2">
    <meta name="author" content="作者名">
    
    <title>ページタイトル | サイト名</title>
    
    <!-- ファビコン -->
    <link rel="icon" href="/favicon.ico">
    <link rel="apple-touch-icon" href="/apple-touch-icon.png">
    
    <!-- CSS -->
    <link rel="stylesheet" href="styles.css">
    
    <!-- Open Graph / Facebook -->
    <meta property="og:type" content="website">
    <meta property="og:url" content="https://example.com/">
    <meta property="og:title" content="ページタイトル">
    <meta property="og:description" content="ページの説明">
    <meta property="og:image" content="https://example.com/image.jpg">
    
    <!-- Twitter -->
    <meta property="twitter:card" content="summary_large_image">
    <meta property="twitter:url" content="https://example.com/">
    <meta property="twitter:title" content="ページタイトル">
    <meta property="twitter:description" content="ページの説明">
    <meta property="twitter:image" content="https://example.com/image.jpg">
</head>
<body>
    <header>
        <nav>
            <!-- ナビゲーション -->
        </nav>
    </header>
    
    <main>
        <!-- メインコンテンツ -->
    </main>
    
    <footer>
        <!-- フッター -->
    </footer>
    
    <!-- JavaScript -->
    <script src="script.js"></script>
</body>
</html>

HTMLタグの記法ルール

タグの基本構文

タグの種類 記法 説明・例
開始タグと終了タグ <tag>内容</tag> 通常の要素。例：<p>段落</p>
空要素（自己完結型） <tag> または <tag /> 内容を持たない要素。例：<br>、<img src="...">
属性付きタグ <tag attribute="value"> 属性を持つ要素。例：<a href="url">
複数属性 <tag attr1="val1" attr2="val2"> スペースで区切る。例：<img src="..." alt="...">
ブール属性 <tag attribute> 値を省略可能。例：<input disabled>
コメント <!-- コメント --> ブラウザに表示されない注釈

属性値の記法

記法 説明 例
ダブルクォート（推奨） 属性値を囲む標準的な方法 class="container"
シングルクォート ダブルクォートの代替 class='container'
クォートなし スペースを含まない単純な値のみ type=text
空の属性値 空文字列を表す alt=""

グローバル属性

すべてのHTML要素で使用可能な属性

属性 説明 使用例
id 要素の一意な識別子 <div id="header">
class CSSクラス名（複数指定可） <div class="container active">
style インラインCSS <p style="color: red;">
title ツールチップテキスト <abbr title="HyperText Markup Language">HTML</abbr>
lang 要素の言語 <p lang="en">Hello</p>
dir テキストの方向 <p dir="rtl">右から左</p>
data-* カスタムデータ属性 <div data-user-id="123">
hidden 要素を非表示にする <div hidden>
tabindex タブ順序を指定 <button tabindex="1">
contenteditable コンテンツを編集可能にする <div contenteditable="true">
spellcheck スペルチェックの有効/無効 <textarea spellcheck="false">
draggable 要素をドラッグ可能にする <img draggable="true">

メタタグの種類

主要なメタタグ一覧

メタタグ 目的 使用例
charset 文字エンコーディング指定 <meta charset="UTF-8">
viewport モバイル表示の制御 <meta name="viewport" content="width=device-width, initial-scale=1.0">
description ページの説明（SEO） <meta name="description" content="ページの説明文">
keywords キーワード（現在は非推奨） <meta name="keywords" content="キーワード1, キーワード2">
author 作者情報 <meta name="author" content="作者名">
robots 検索エンジンへの指示 <meta name="robots" content="index, follow">
refresh 自動リフレッシュ/リダイレクト <meta http-equiv="refresh" content="5;url=https://example.com">
theme-color ブラウザのテーマカラー <meta name="theme-color" content="#7f6df2">

リンクタイプ（rel属性）

<link>タグと<a>タグで使用するrel属性値

rel属性値 説明 使用例
stylesheet 外部CSSファイル <link rel="stylesheet" href="style.css">
icon ファビコン <link rel="icon" href="favicon.ico">
preconnect 外部ドメインへの事前接続 <link rel="preconnect" href="https://fonts.googleapis.com">
preload リソースの事前読み込み <link rel="preload" href="font.woff2" as="font">
prefetch 将来使用するリソースの先読み <link rel="prefetch" href="next-page.html">
canonical 正規URLの指定 <link rel="canonical" href="https://example.com/page">
alternate 代替バージョン <link rel="alternate" hreflang="en" href="/en/page">
nofollow リンクをたどらない（<a>タグ） <a href="..." rel="nofollow">
noopener 新規ウィンドウのセキュリティ <a href="..." target="_blank" rel="noopener">
noreferrer リファラー情報を送らない <a href="..." rel="noreferrer">

イベント属性

よく使うイベントハンドラ属性

イベント属性 発生タイミング 使用例
onclick クリック時 <button onclick="alert('クリック')">
onchange 値が変更された時 <input onchange="handleChange()">
onsubmit フォーム送信時 <form onsubmit="return validate()">
onload 読み込み完了時 <body onload="init()">
onmouseover マウスが乗った時 <div onmouseover="highlight()">
onmouseout マウスが離れた時 <div onmouseout="unhighlight()">
onfocus フォーカスを得た時 <input onfocus="showHint()">
onblur フォーカスを失った時 <input onblur="hideHint()">
onkeydown キーを押した時 <input onkeydown="checkKey(event)">
onkeyup キーを離した時 <input onkeyup="search()">

ARIA属性

アクセシビリティ向上のためのARIA属性

ARIA属性 説明 使用例
aria-label 要素のラベルを提供 <button aria-label="閉じる">×</button>
aria-labelledby ラベル要素のIDを参照 <nav aria-labelledby="nav-title">
aria-describedby 説明要素のIDを参照 <input aria-describedby="email-hint">
aria-hidden スクリーンリーダーから隠す <span aria-hidden="true">装飾</span>
aria-live 動的コンテンツの通知 <div aria-live="polite">更新内容</div>
aria-required 必須フィールドを示す <input aria-required="true">
aria-invalid 無効な入力を示す <input aria-invalid="true">
aria-expanded 展開/折りたたみ状態 <button aria-expanded="false">
aria-current 現在のアイテムを示す <a aria-current="page">現在のページ</a>
role 要素の役割を定義 <div role="navigation">
