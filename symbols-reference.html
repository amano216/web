Web開発で使う記号の読み方と使い方

HTMLやCSS、JavaScriptでよく使用される記号の読み方と、それぞれの使い方をまとめました。

基本的な記号

HTMLで使う記号

記号 英語の読み方 日本語の読み方・使い方
< Less Than / Left Angle Bracket 小なり（しょうなり）/ 左山括弧（ひだりやまかっこ）。HTMLタグの開始を示す。例：<div>
> Greater Than / Right Angle Bracket 大なり（だいなり）/ 右山括弧（みぎやまかっこ）。HTMLタグの終了を示す。例：</div>
/ Slash / Forward Slash スラッシュ。終了タグや自己完結型タグで使用。例：</p>、<br />
= Equals Sign イコール / 等号（とうごう）。属性値の代入に使用。例：class="container"
" Double Quote ダブルクォート / 二重引用符（にじゅういんようふ）。属性値を囲む。例：href="index.html"
' Single Quote シングルクォート / 単一引用符（たんいついんようふ）。属性値を囲む（ダブルクォートの代替）。
& Ampersand アンパサンド。HTMLエンティティの開始。例：&lt;（<を表示）
! Exclamation Mark エクスクラメーション / 感嘆符（かんたんふ）。DOCTYPE宣言やコメントで使用。例：<!DOCTYPE html>
- Hyphen / Dash ハイフン。HTMLコメントで使用。例：<!-- コメント -->

CSSで使う記号

記号 英語の読み方 日本語の読み方・使い方
{ Left Curly Brace 左波括弧（ひだりなみかっこ）/ 左中括弧。CSSルールの開始。例：.class { }
} Right Curly Brace 右波括弧（みぎなみかっこ）/ 右中括弧。CSSルールの終了。
: Colon コロン。プロパティと値の区切り。例：color: red;
; Semicolon セミコロン。CSS宣言の終了。例：margin: 10px;
. Period / Dot ピリオド / ドット。クラスセレクタ。例：.container
# Hash / Pound Sign ハッシュ / シャープ。IDセレクタや色コード。例：#header、#ff0000
* Asterisk アスタリスク。全称セレクタ。例：* { margin: 0; }
, Comma カンマ / コンマ。複数セレクタの区切り。例：h1, h2, h3 { }
> Greater Than 大なり。子セレクタ。例：div > p（直接の子要素）
+ Plus Sign プラス。隣接兄弟セレクタ。例：h1 + p（直後の要素）
~ Tilde チルダ。一般兄弟セレクタ。例：h1 ~ p（後続の兄弟要素）
@ At Sign アットマーク。アットルール。例：@media、@import
! Exclamation Mark エクスクラメーション。!importantで使用。例：color: red !important;
() Parentheses 丸括弧（まるかっこ）。関数や値のグループ化。例：rgb(255, 0, 0)
[] Square Brackets 角括弧（かくかっこ）。属性セレクタ。例：input[type="text"]

URLやパスで使う記号

記号 英語の読み方 日本語の読み方・使い方
/ Slash スラッシュ。ディレクトリの区切り。例：/images/logo.png
. Dot / Period ドット。ファイル拡張子の前。例：style.css
.. Double Dot ダブルドット。親ディレクトリ。例：../images/
? Question Mark クエスチョンマーク。URLパラメータの開始。例：page.html?id=123
& Ampersand アンパサンド。URLパラメータの連結。例：?id=123&name=test
# Hash ハッシュ。アンカーリンク。例：page.html#section1
: Colon コロン。プロトコルの区切り。例：https://example.com
// Double Slash ダブルスラッシュ。プロトコル相対URL。例：//cdn.example.com

特殊な記号と用途

記号 英語の読み方 日本語の読み方・使い方
_ Underscore アンダースコア / アンダーバー。命名規則で単語を繋ぐ。例：user_name
- Hyphen / Dash ハイフン。ケバブケースで使用。例：main-content
| Pipe / Vertical Bar パイプ / 縦棒（たてぼう）。CSS属性セレクタで使用。例：[lang|="en"]
^ Caret キャレット。CSS属性セレクタで「〜で始まる」。例：[href^="https"]
$ Dollar Sign ドル記号。CSS属性セレクタで「〜で終わる」。例：[href$=".pdf"]
% Percent Sign パーセント。相対的な値の単位。例：width: 50%;
\ Backslash バックスラッシュ / 円記号。エスケープ文字。例：content: "\"引用\"";
` Backtick / Grave Accent バッククォート。JavaScriptのテンプレートリテラル。例：`Hello ${name}`

記号の組み合わせ

よく使われる記号の組み合わせ

記号の組み合わせ 英語の説明 日本語の説明・使用例
<!-- --> HTML Comment HTMLコメント。ブラウザに表示されない注釈。例：<!-- ナビゲーション開始 -->
/* */ CSS Comment CSSコメント。スタイルシート内の注釈。例：/* メインカラー */
:: Double Colon ダブルコロン。CSS疑似要素。例：::before、::after
:: Scope Resolution スコープ解決演算子（他の言語）。CSSでは疑似要素を示す。
</> Self-closing Tag 自己完結型タグ。例：<img />、<br />
<?> Processing Instruction 処理命令（PHPなど）。例：<?php echo "Hello"; ?>
=> Arrow Function アロー関数（JavaScript）。例：() => { }
!== Strict Inequality 厳密不等価演算子（JavaScript）。型も含めて比較。
=== Strict Equality 厳密等価演算子（JavaScript）。型も含めて比較。

エスケープが必要な記号

HTMLで表示する際の特殊文字

表示したい記号 HTMLエンティティ 説明・用途
< &lt; 小なり記号を表示。HTMLタグと区別するため。
> &gt; 大なり記号を表示。HTMLタグと区別するため。
& &amp; アンパサンドを表示。エンティティと区別するため。
" &quot; ダブルクォートを表示。属性値内で使用時。
' &apos; シングルクォートを表示。属性値内で使用時。
© &copy; 著作権記号。例：&copy; 2024
® &reg; 登録商標記号。
™ &trade; 商標記号。
&nbsp; 改行されない空白（Non-Breaking Space）。

プログラミングでよく使う記号の読み方

プログラマーが使う特別な読み方

記号 一般的な読み方 プログラマーの読み方・俗称
! エクスクラメーション 「ビックリ」「バン（bang）」。例：「!important」を「バン・インポータント」
# シャープ / ハッシュ 「ナンバー」「ハッシュ」。例：「#header」を「ハッシュ・ヘッダー」
* アスタリスク 「スター」「ワイルドカード」。全称セレクタは「スター・セレクタ」
& アンパサンド 「アンド」。「&&」は「アンド・アンド」
| パイプ 「オア」「パイプ」。「||」は「オア・オア」
^ キャレット 「ハット」。正規表現で行頭を表す。
$ ドル 「ダラー」。正規表現で行末を表す。
~ チルダ 「ニョロ」「チルダ」。ホームディレクトリを表すことも。
