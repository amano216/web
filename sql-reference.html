SQLコマンド一覧

データベース操作に使用される主要なSQLコマンドを、カテゴリ別にまとめました。

データ定義言語（DDL）

テーブル操作

コマンド 英語の意味 (Meaning) 日本語の意味・用途
CREATE TABLE Create Table 新しいテーブルを作成する。列名とデータ型を定義
ALTER TABLE Alter Table 既存のテーブル構造を変更する。列の追加・削除・変更
DROP TABLE Drop Table テーブルを完全に削除する。データも含めて削除
TRUNCATE TABLE Truncate Table テーブルのデータをすべて削除する。構造は保持
RENAME TABLE Rename Table テーブル名を変更する

列の定義と制約

制約・キーワード 英語の意味 (Meaning) 日本語の意味・用途
PRIMARY KEY Primary Key 主キー。レコードを一意に識別する列を指定
FOREIGN KEY Foreign Key 外部キー。他のテーブルとの関連を定義
UNIQUE Unique 一意制約。重複した値を許可しない
NOT NULL Not Null NULL値を許可しない制約
DEFAULT Default デフォルト値を設定する
CHECK Check 値の条件をチェックする制約
AUTO_INCREMENT Auto Increment 自動的に連番を生成する（MySQL）

インデックス操作

コマンド 英語の意味 (Meaning) 日本語の意味・用途
CREATE INDEX Create Index インデックスを作成する。検索速度を向上させる
DROP INDEX Drop Index インデックスを削除する
CREATE UNIQUE INDEX Create Unique Index 一意インデックスを作成する。重複を防ぐ

データ操作言語（DML）

データの取得

コマンド 英語の意味 (Meaning) 日本語の意味・用途
SELECT Select データを取得する。最も基本的なクエリ
FROM From データを取得するテーブルを指定する
WHERE Where 条件を指定してデータを絞り込む
ORDER BY Order By 結果を並び替える。昇順（ASC）・降順（DESC）
GROUP BY Group By データをグループ化する。集計関数と併用
HAVING Having GROUP BY後の条件を指定する
LIMIT Limit 取得するレコード数を制限する
OFFSET Offset 取得開始位置を指定する。ページング処理で使用
DISTINCT Distinct 重複を除いた結果を取得する

データの追加・更新・削除

コマンド 英語の意味 (Meaning) 日本語の意味・用途
INSERT INTO Insert Into 新しいレコードを追加する
VALUES Values 追加するデータの値を指定する
UPDATE Update 既存のレコードを更新する
SET Set 更新する列と値を指定する
DELETE FROM Delete From レコードを削除する。WHERE句なしは全削除

結合（JOIN）操作

テーブルの結合

結合タイプ 英語の意味 (Meaning) 日本語の意味・用途
INNER JOIN Inner Join 内部結合。両方のテーブルに存在するデータのみ取得
LEFT JOIN Left Join 左外部結合。左テーブルのすべてと右テーブルの一致データ
RIGHT JOIN Right Join 右外部結合。右テーブルのすべてと左テーブルの一致データ
FULL OUTER JOIN Full Outer Join 完全外部結合。両方のテーブルのすべてのデータ
CROSS JOIN Cross Join クロス結合。すべての組み合わせを生成
SELF JOIN Self Join 自己結合。同じテーブルを結合
ON On 結合条件を指定する

集計関数

データの集計

関数 英語の意味 (Meaning) 日本語の意味・用途
COUNT() Count レコード数を数える。NULL値は除外
SUM() Sum 合計値を計算する
AVG() Average 平均値を計算する
MAX() Maximum 最大値を取得する
MIN() Minimum 最小値を取得する
GROUP_CONCAT() Group Concatenate グループ内の値を連結する（MySQL）

サブクエリと共通テーブル式

サブクエリ

キーワード 英語の意味 (Meaning) 日本語の意味・用途
IN In リスト内の値と一致するかチェック
NOT IN Not In リスト内の値と一致しないかチェック
EXISTS Exists サブクエリが結果を返すかチェック
NOT EXISTS Not Exists サブクエリが結果を返さないかチェック
ANY Any いずれかの値と比較
ALL All すべての値と比較

共通テーブル式（CTE）

キーワード 英語の意味 (Meaning) 日本語の意味・用途
WITH With 共通テーブル式を定義する。一時的な名前付き結果セット
RECURSIVE Recursive 再帰的なCTEを定義する。階層データの処理

条件式と演算子

比較演算子

演算子 英語の意味 (Meaning) 日本語の意味・用途
= Equal 等しい
!= or <> Not Equal 等しくない
> Greater Than より大きい
< Less Than より小さい
>= Greater Than or Equal 以上
<= Less Than or Equal 以下
BETWEEN Between 範囲内かチェック
LIKE Like パターンマッチング。ワイルドカード使用可
IS NULL Is Null NULL値かチェック
IS NOT NULL Is Not Null NULL値でないかチェック

論理演算子

演算子 英語の意味 (Meaning) 日本語の意味・用途
AND And すべての条件が真の場合に真
OR Or いずれかの条件が真の場合に真
NOT Not 条件を否定する

文字列関数

文字列操作

関数 英語の意味 (Meaning) 日本語の意味・用途
CONCAT() Concatenate 文字列を連結する
LENGTH() Length 文字列の長さを取得する
UPPER() Upper 大文字に変換する
LOWER() Lower 小文字に変換する
TRIM() Trim 前後の空白を削除する
SUBSTRING() Substring 部分文字列を取得する
REPLACE() Replace 文字列を置換する

日付・時刻関数

日付操作

関数 英語の意味 (Meaning) 日本語の意味・用途
NOW() Now 現在の日時を取得する
CURRENT_DATE Current Date 現在の日付を取得する
CURRENT_TIME Current Time 現在の時刻を取得する
DATE_FORMAT() Date Format 日付を指定形式で表示する（MySQL）
DATE_ADD() Date Add 日付に期間を加算する
DATE_SUB() Date Subtract 日付から期間を減算する
DATEDIFF() Date Difference 日付の差を計算する

トランザクション制御

トランザクション管理

コマンド 英語の意味 (Meaning) 日本語の意味・用途

ROLLBACK Rollback トランザクションを取り消す。変更を破棄
SAVEPOINT Savepoint トランザクション内の保存ポイントを作成
SET AUTOCOMMIT Set Autocommit 自動コミットの有効・無効を設定

データベース操作
