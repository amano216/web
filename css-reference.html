CSSプロパティ総合リファレンス

Webページのスタイリングに使用される主要なCSSプロパティを、カテゴリごとにまとめました。

CSSプロパティ一覧

テキストとフォント

文字の見た目を制御するプロパティです。

プロパティ 英語の意味 (Meaning) 日本語の意味・用途
color Color 文字色を指定。例：color: #333;、color: red;、color: rgb(255, 0, 0);
font-size Font Size 文字の大きさを指定。例：font-size: 16px;、font-size: 1.2rem;
font-family Font Family フォントの種類を指定。例：font-family: Arial, sans-serif;
font-weight Font Weight 文字の太さを指定。例：font-weight: bold;、font-weight: 700;
font-style Font Style 文字のスタイル（斜体など）を指定。例：font-style: italic;
text-align Text Alignment テキストの水平方向の配置。例：text-align: center;、text-align: right;
text-decoration Text Decoration テキストの装飾（下線など）。例：text-decoration: underline;、text-decoration: none;
line-height Line Height 行の高さを指定。例：line-height: 1.5;、line-height: 24px;
letter-spacing Letter Spacing 文字間隔を調整。例：letter-spacing: 0.1em;
text-transform Text Transform 大文字・小文字の変換。例：text-transform: uppercase;

ボックスモデル

要素のサイズや余白を制御するプロパティです。

プロパティ 英語の意味 (Meaning) 日本語の意味・用途
width Width 要素の幅を指定。例：width: 300px;、width: 50%;、width: 100vw;
height Height 要素の高さを指定。例：height: 200px;、height: auto;
margin Margin 要素の外側の余白。例：margin: 20px;、margin: 10px 20px;
padding Padding 要素の内側の余白。例：padding: 15px;、padding: 10px 20px 30px 40px;
border Border 要素の境界線。例：border: 1px solid #000;、border-radius: 5px;
box-sizing Box Sizing 幅と高さの計算方法。例：box-sizing: border-box;（paddingとborderを含む）
overflow Overflow 要素からはみ出た内容の処理。例：overflow: hidden;、overflow: scroll;
min-width Minimum Width 最小幅を指定。例：min-width: 200px;
max-width Maximum Width 最大幅を指定。例：max-width: 1200px;

背景とカラー

背景色や背景画像を制御するプロパティです。

プロパティ 英語の意味 (Meaning) 日本語の意味・用途
background-color Background Color 背景色を指定。例：background-color: #f0f0f0;
background-image Background Image 背景画像を指定。例：background-image: url('image.jpg');
background-size Background Size 背景画像のサイズ。例：background-size: cover;、background-size: contain;
background-position Background Position 背景画像の位置。例：background-position: center center;
background-repeat Background Repeat 背景画像の繰り返し。例：background-repeat: no-repeat;
background Background Shorthand 背景関連の一括指定。例：background: #fff url('bg.jpg') no-repeat center;
opacity Opacity 要素の不透明度。例：opacity: 0.8;（0は完全透明、1は完全不透明）
background-attachment Background Attachment 背景画像の固定方法。例：background-attachment: fixed;

レイアウト - ポジショニング

要素の配置方法を制御するプロパティです。

プロパティ 英語の意味 (Meaning) 日本語の意味・用途
display Display 要素の表示形式。例：display: block;、display: inline;、display: none;
position Position 要素の配置方法。例：position: relative;、position: absolute;、position: fixed;
top Top 上からの位置を指定。例：top: 10px;（positionと併用）
right Right 右からの位置を指定。例：right: 20px;
bottom Bottom 下からの位置を指定。例：bottom: 0;
left Left 左からの位置を指定。例：left: 50px;
z-index Z-axis Index 重なり順序を指定。例：z-index: 10;（大きい値が前面）
float Float 要素を左右に浮かせる。例：float: left;（現在はFlexboxやGridを推奨）
clear Clear floatの回り込みを解除。例：clear: both;

Flexbox（フレキシブルボックス）

柔軟なレイアウトを作成するための最新の手法です。

プロパティ 英語の意味 (Meaning) 日本語の意味・用途
display: flex Flex Container Flexコンテナを作成。子要素が横並びになる。
flex-direction Flex Direction 主軸の方向。例：flex-direction: row;（横）、flex-direction: column;（縦）
justify-content Justify Content 主軸方向の配置。例：justify-content: center;、justify-content: space-between;
align-items Align Items 交差軸方向の配置。例：align-items: center;、align-items: flex-start;
flex-wrap Flex Wrap 折り返しの設定。例：flex-wrap: wrap;（折り返す）、flex-wrap: nowrap;
gap Gap アイテム間の隙間。例：gap: 20px;、gap: 10px 20px;
flex Flex Shorthand 伸縮性の一括指定。例：flex: 1;（均等に伸びる）、flex: 0 0 200px;
align-self Align Self 個別アイテムの交差軸配置。例：align-self: flex-end;
order Order 表示順序を変更。例：order: 2;（デフォルトは0）

CSS Grid（グリッドレイアウト）

2次元のグリッドレイアウトを作成するための強力な手法です。

プロパティ 英語の意味 (Meaning) 日本語の意味・用途
display: grid Grid Container グリッドコンテナを作成。
grid-template-columns Grid Template Columns 列の定義。例：grid-template-columns: 1fr 2fr 1fr;、repeat(3, 1fr)
grid-template-rows Grid Template Rows 行の定義。例：grid-template-rows: 100px auto 50px;
grid-gap Grid Gap グリッドの隙間。例：grid-gap: 20px;（現在はgapを推奨）
grid-column Grid Column アイテムの列位置。例：grid-column: 1 / 3;（1列目から3列目まで）
grid-row Grid Row アイテムの行位置。例：grid-row: 2 / 4;
grid-area Grid Area 名前付きエリアの指定。例：grid-area: header;
grid-template-areas Grid Template Areas エリアの名前付けレイアウト。例："header header" "main sidebar"
justify-items Justify Items セル内の水平配置。例：justify-items: center;
align-items Align Items セル内の垂直配置。例：align-items: start;

トランジションとアニメーション

要素の動きや変化を制御するプロパティです。

プロパティ 英語の意味 (Meaning) 日本語の意味・用途
transition Transition プロパティ変化のアニメーション。例：transition: all 0.3s ease;
transition-property Transition Property 対象プロパティ。例：transition-property: background-color, transform;
transition-duration Transition Duration 変化にかかる時間。例：transition-duration: 0.5s;
transition-timing-function Timing Function 変化の速度曲線。例：transition-timing-function: ease-in-out;
transform Transform 要素の変形。例：transform: rotate(45deg);、transform: scale(1.2);
animation Animation キーフレームアニメーション。例：animation: slide 2s infinite;
@keyframes Keyframes アニメーションの定義。例：@keyframes slide { from { left: 0; } to { left: 100px; } }
animation-duration Animation Duration アニメーションの長さ。例：animation-duration: 3s;
animation-iteration-count Iteration Count 繰り返し回数。例：animation-iteration-count: infinite;

疑似クラスと疑似要素

要素の特定の状態や部分をスタイリングします。

セレクタ 英語の意味 (Meaning) 日本語の意味・用途
:hover Hover State マウスが乗った時の状態。例：a:hover { color: red; }
:active Active State クリック中の状態。例：button:active { background: #ccc; }
:focus Focus State フォーカスが当たった状態。例：input:focus { border-color: blue; }
:first-child First Child 最初の子要素。例：li:first-child { font-weight: bold; }
:last-child Last Child 最後の子要素。例：p:last-child { margin-bottom: 0; }
:nth-child(n) Nth Child n番目の子要素。例：tr:nth-child(even) { background: #f0f0f0; }
::before Before Pseudo-element 要素の前に内容を挿入。例：h2::before { content: "▶ "; }
::after After Pseudo-element 要素の後に内容を挿入。例：.required::after { content: " *"; color: red; }
::first-line First Line 最初の行のスタイル。例：p::first-line { font-weight: bold; }
::selection Selection 選択されたテキスト。例：::selection { background: yellow; }

レスポンシブデザイン

デバイスのサイズに応じてスタイルを変更します。

機能 英語の意味 (Meaning) 日本語の意味・用途
@media Media Query 条件に応じたスタイル適用。例：@media (max-width: 768px) { ... }
min-width Minimum Width 最小幅以上の場合に適用。例：@media (min-width: 1024px)
max-width Maximum Width 最大幅以下の場合に適用。例：@media (max-width: 767px)
orientation Orientation 画面の向き。例：@media (orientation: landscape)（横向き）
aspect-ratio Aspect Ratio 画面の縦横比。例：@media (aspect-ratio: 16/9)
prefers-color-scheme Color Scheme Preference ダークモード対応。例：@media (prefers-color-scheme: dark)

CSS変数（カスタムプロパティ）

再利用可能な値を定義します。

機能 英語の意味 (Meaning) 日本語の意味・用途
--variable-name Custom Property Definition 変数の定義。例：--primary-color: #3498db;
var() Variable Usage 変数の使用。例：color: var(--primary-color);
:root Root Element グローバル変数の定義場所。例：:root { --spacing: 20px; }
var(--name, fallback) Variable with Fallback フォールバック値付き。例：color: var(--text-color, #000);

その他の重要なプロパティ

プロパティ 英語の意味 (Meaning) 日本語の意味・用途
cursor Cursor マウスカーソルの形状。例：cursor: pointer;、cursor: not-allowed;
visibility Visibility 要素の表示/非表示（スペースは維持）。例：visibility: hidden;
box-shadow Box Shadow ボックスの影。例：box-shadow: 0 2px 5px rgba(0,0,0,0.3);
text-shadow Text Shadow テキストの影。例：text-shadow: 2px 2px 4px rgba(0,0,0,0.5);
filter Filter 画像効果。例：filter: blur(5px);、filter: grayscale(100%);
pointer-events Pointer Events マウスイベントの制御。例：pointer-events: none;（クリック無効）
user-select User Select テキスト選択の制御。例：user-select: none;（選択不可）
white-space White Space 空白文字の処理。例：white-space: nowrap;（改行しない）
word-break Word Break 単語の改行ルール。例：word-break: break-all;（どこでも改行可）
object-fit Object Fit 画像・動画のサイズ調整。例：object-fit: cover;、object-fit: contain;

CSSのベストプラクティス

効率的なCSSの書き方

プラクティス 英語の意味 (Meaning) 日本語の意味・用途
モバイルファースト Mobile First 小さい画面用のスタイルを先に書き、メディアクエリで大画面対応を追加。
BEM命名規則 Block Element Modifier クラス名の構造化。例：.card__title--large
CSS変数の活用 CSS Variables 色やサイズを変数化して管理。保守性が向上。
セレクタの詳細度 Specificity IDセレクタは避け、クラスセレクタを中心に使用。
プロパティの順序 Property Order レイアウト → ボックスモデル → 装飾の順で記述。
短縮プロパティ Shorthand Properties margin: 10px 20px;のように一括指定を活用。
リセットCSS CSS Reset ブラウザのデフォルトスタイルをリセット。
コメントの活用 Comments セクションごとにコメントで区切り、可読性を向上。
